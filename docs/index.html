FirstStrike.css

A lightweight & modern alternative to normalize.css

FirstStrike.css builds off of the widely used Normalize.css, making it more modern and lightweight.

How? Well a lot of time has passed since Normalize.css stabilized into what it is today, and many issues and inconsistencies in browsers have been fixed!

FirstStrike.css removes code for issues that have been long fixed in the majority of browsers. And yes, that includes dropping support for Internet Explorer.

Overall this makes for an even more minimal reset and smaller filesize. While it might not be much, small savings can add up!

Download

npm install astraloverflow/firststrike.css
